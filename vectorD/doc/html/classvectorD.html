template<typename T>
class vectorD< T >

T.D.A. vectorD.

Una instancia c del tipo de datos abstracto vectorD es un objeto, el cual es un conjunto de elementos ordenados. El vector vendra dado por un valor nulo, quedando asi el contenido del vector "dividido" en dos, podiendo iterar sobre el vector de dos formas: Sobre su contenido no nulo, y sobre todo su contenido.

Lo representamos:

v[n_ele] = {elem, elem, elem... (n_ele)...}
